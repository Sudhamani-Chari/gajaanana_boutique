# Gajaanana Boutique

Welcome to Gajaanana Boutique, a fashion boutique website.

## Description

This is a simple static website for Gajaanana Boutique, featuring sections for home, about, products, and contact information.

## Files

- `index.html`: Main HTML file
- `styles.css`: CSS styles
- `script.js`: JavaScript for interactivity

## Usage

Open `index.html` in a web browser to view the website.

## Development

To run a local server, you can use Python's built-in server:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.
